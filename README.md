### DIY 使用指南

#### 更换键帽

- **拆卸**：直接拔下键帽即可。
- **安装**：注意键帽十字槽的方向——上下方向与左右方向的宽度不同，安装时**切勿将键帽旋转 90°**，否则会卡死或损伤轴体与键帽。

#### 更换轴体

- **拆卸**：使用拔轴器（可能需自行购买）卡住轴体两侧的卡扣，将轴体垂直向上拔出。
- **安装**：
  1. 先检查按键轴引脚是否笔直、垂直，避免歪斜。
  2. 注意引脚对准插孔方向，垂直对准后再下压。
  3. 均匀用力、轻轻压入到位。
- **重要提示**：
  - 轴体不可倾斜压下，否则会损坏轴体、插座或 PCB。
  - 不可使用暴力硬压，必要时先取下键帽，再处理轴体。

### 使用与保养建议

- 进行键帽、轴体更换前，建议先断开键盘与主机的连接，避免误触发。
- 拔轴器推荐另行选购专用型号，避免使用螺丝刀等工具强行撬动。
- 模块磁吸面应保持洁净，避免金属碎屑、油污影响吸附与信号接触。
- 不使用时可单独存放模块，避免长期重压。
